🎓 Cómo funciona el simulador

Este simulador te permite experimentar con operaciones reales de comercio exterior ecuatoriano:

  1. Selecciona productos para importar o exportar
  2. Revisa la clasificación arancelaria y documentos requeridos
  3. Calcula automáticamente todos los impuestos
  4. Define tu precio de venta y verifica utilidades
  5. Responde a eventos aleatorios que afectan el mercado
  6. Sube de nivel desbloqueando productos más complejos

📊 Fórmulas de Cálculo Ecuador

CIF = FOB + Flete + Seguro (1% FOB)

Base Imponible = CIF

Advalorem = CIF × % Arancel

FODINFA = CIF × 0.5%

Base IVA = CIF + Advalorem + FODINFA + ICE

IVA = Base IVA × 15%

Total Importación = CIF + Advalorem + FODINFA + ICE + IVA

📋 Documentos de Control Previo

Algunos productos requieren permisos especiales antes de importar:

  • Ministerio de Defensa: Armas, municiones, explosivos
  • Ministerio de Agricultura: Productos agrícolas, animales vivos
  • Ministerio de Salud: Medicinas, cosméticos, alimentos procesados
  • INEN: Normas técnicas para ciertos productos industriales
